# Ireland Employment Permit Tracker

Interactive dashboard for Ireland employment permit statistics (2010–2026), built from the
[Department of Enterprise, Trade & Employment](https://enterprise.gov.ie/en/publications/) publications.

- **Live dashboard:** https://vivek19398.github.io/csep_permit_tracker/
- `permits-by-nationality-2010-2026.csv` — Year, Nationality, New, Renewal, Issued, Refused, Withdrawn
- `permits-by-county-2010-2026.csv` — Year, County, …
- `permits-by-sector-2010-2026.csv` — Year, Month, Sector, …
- `permits-issued-to-companies-2010-2026.csv` — Year, Employer Name, Total
- `raw-downloads/` — original xlsx files from enterprise.gov.ie + the merge script

2026 is year-to-date (Jan–May). "Issued" = "Total" in the 2010–2019 publications.
